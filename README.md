# README
[MUSCLE3](https://muscle3.readthedocs.io/en/latest/) wrapper for [TORAX](https://github.com/google-deepmind/torax/), with serialized [IMAS](https://imas-python.readthedocs.io/en/stable/) I/O.

# Documentation
Documentation is autogenerated from the source using [Sphinx](http://sphinx-doc.org/).
The documentation can be manually generated by installing sphinx and running:

```bash
make -C docs html
```
